# Script Template — History & Mystery Short

Copy this block per video. Fill the brackets, delete the guidance, keep the
constraints. Target **110–140 words, ~45–50s.** Worked example:
[`package_template.json`](package_template.json) (the Dancing Plague of 1518).

## Script

> **[0–2s] HARD HOOK** — ≤5 words, ends in `?` or `!`. A disbelief beat, not the
> claim itself.
> e.g. `This actually happened?!` / `They weaponized BEES?` / `No one survived?!`

[HOOK]

> **[2–8s] SETUP** — drop the real claim now. Name the **year, place, people**.
> Specificity = credibility.

[In <YEAR>, in <PLACE>, <WHO> <did WHAT>...]

> **[8–35s] ESCALATING FACTS** — 6–9 facts, each more "no way" than the last.
> Digits not number-words. Use "records show" for confirmed facts.

[FACT 1 — the setup pays off]
[FACT 2 — bigger]
[FACT 3 — bigger still]
[FACT 4–6 — keep raising the disbelief]

> **[35–50s] TWIST / REVEAL** — the detail that flips it, or the unsolved core.
> If it's contested, use "one theory says…" / "another says…". Never "this proves".

[TWIST]

> **[final] PAYOFF / KICKER** — last sentence ends in `?` AND names something
> specific from the story. Make it answerable + a little divisive (drives
> comments). Never generic "what do you think?".

[So <specific question naming an entity>?]

## Pre-flight checklist (must all be true)
- [ ] 110–140 words.
- [ ] Hook ≤5 words, ends `?`/`!`.
- [ ] Kicker ends `?`, names a story entity.
- [ ] Numbers written as **digits** ("1518", "400", "3 days").
- [ ] Confirmed facts vs. theories labeled per the fact-check file.
- [ ] No banned slop phrases (run `slop_check.py`).
- [ ] Every fact has a real source logged.
- [ ] Unsolved parts left explicitly unsolved.
