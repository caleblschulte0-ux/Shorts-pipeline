Reaction-emoji overlay set (skull/fire/sob/joy/eyes/mindblown/scream/
flushed/pleading/rage .png)

Current assets: Microsoft Fluent UI Emoji, 3D style (256x256 RGBA).
Source:  https://github.com/microsoft/fluentui-emoji
License: MIT (Copyright (c) Microsoft Corporation) — free commercial use.
Mapping note: rage.png is Fluent's "Pouting face" 3D asset.

Chosen per the channel playbook §11: one cohesive, dimensional, glossy,
iOS-style set — never mixed with flat/native/sticker emoji.

Fallback generator: make_emoji.py renders the same names from Noto Color
Emoji (OFL 1.1) if the Fluent assets ever need regenerating offline. The
committed PNGs are what the renderer uses.
