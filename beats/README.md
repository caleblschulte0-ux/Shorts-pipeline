# Beats workspace — "After You've Gone" flip

Kanye-style sampling sketches built from `source/After_Youve_Gone.mp3`.

## Layout
- `source/`   — original song(s)
- `stems/`    — Demucs-separated stems (vocals/bass/drums/other)
- `kit/`      — drum one-shots
- `pipeline/` — render scripts
- `renders/`  — exported beats (wav/mp3)

## Direction
Pay homage to the original without being "just the jazz record." Lift the
**vocal** ("windows down, singing in summer"), tame the brass. Target lane:
Gold Digger / Good Life / Famous — bright, celebratory, vocal-chop-forward,
summery, punchy but uncluttered.

## Method
1. Demucs stem-separate -> isolate vocals (+ light "other" for warmth), drop brass.
2. Chipmunk-soul the vocal: resample up (pitch+speed) for that sped-up shimmer.
3. Loop a clean chop; lay punchy summery drums + moving sub/808.
4. Mix with room for a rap vocal; warm/bright master.
