# Shorts Pipeline

An automated short-form video pipeline that posts daily news explainer
videos to **YouTube** and other social platforms.

## About

Shorts Pipeline is a personal automation tool that discovers trending
news from public RSS sources, generates 60-80 word explainer scripts,
renders them into vertical 1080×1920 video format, and uploads the
finished videos to the operator's **YouTube** channel via the
**YouTube Data API** under the operator's own authorization. No end
users sign in to the service; it operates only on a single channel
owned by the operator.

The service uses **YouTube API Services** to upload videos and read
back public statistics. By using this service the operator agrees to
the [YouTube Terms of Service](https://www.youtube.com/t/terms) and
acknowledges the
[Google Privacy Policy](https://policies.google.com/privacy).
